# wormhole-cctp

To install dependencies:

```bash
bun install
```

To run:

```bash
cp .env.example .env
```

Add your private key to the `.env` file.

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.0.23. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
